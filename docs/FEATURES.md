# Features Documentation

## User Roles

### 1. Site Admin
- Manage all users
- Approve/reject locations, events, packages
- View all callback requests
- Access admin dashboard

### 2. Government Department
- Create and manage locations
- Approve content for their assigned state
- Upload accommodations
- View analytics

### 3. Tourist Guide
- Create travel packages
- Create events
- Manage callback requests
- Submit bids for group travel

### 4. Regular User
- Browse locations, events, packages
- Request callbacks
- Post in community
- Express interest in packages/events

## Core Features

### Locations Management
- Browse travel destinations by country/state/area
- View detailed location information
- See attractions and how to reach
- Admin: Create, edit, approve locations

### Events & Festivals
- Discover local events and festivals
- Filter by date and location
- Request callbacks from event hosts
- Admin: Manage event types and approvals

### Travel Packages
- Browse curated travel packages
- View detailed itineraries
- Request callbacks for bookings
- Admin: Manage package approvals

### Community
- Share travel experiences with photos/videos
- Like and comment on posts
- Follow other travelers
- Location-tagged posts

### Accommodations
- Browse hotels, restaurants, resorts
- Government-approved listings
- Contact information and images
- Admin: Upload and manage listings

### Callback System
- Users can request callbacks for events/packages
- Hosts receive notifications
- Track contacted status
- Admin can view all requests

### Group Travel
- Create group travel requests
- Tourist guides submit bids
- View detailed itineraries and costs
- Express interest in group trips

## Admin Features

### Dashboard
- Overview of pending approvals
- User statistics
- Content metrics

### User Management
- View all users by role
- Activate/deactivate accounts
- Assign states to government departments

### Approval Queue
- Review pending locations, events, packages
- Approve or reject with reasons
- Track approval history

### Callback Management
- View all callback requests
- Filter by event/package
- Mark as contacted
- Export data
